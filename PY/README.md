# Create a virtual environment:
Windows:
```
py -m venv .venv
```
Unix/macOS:
```
python3 -m venv .venv
```


# Activate a virtual environment:
Windows:
```
.venv\Scripts\activate
or deactivate
```
Unix/macOS:
```
source .venv/bin/activate
```

# Install the required packages
```
pip install -r requirements.txt
```
